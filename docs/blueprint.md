# **App Name**: Studio Noir

## Core Features:

- Portfolio Gallery: Display the media studio's work in a visually appealing gallery format.
- Project Details: Provide detailed information about each project, including description, media and team members.
- Services Offered: Showcase the media studio's list of services, complete with brief descriptions and visuals.
- About Us Section: Introduce the studio's team members and company vision.
- Contact Form: Enable potential clients to directly reach out to the studio, complete with form validation and submission confirmation.

## Style Guidelines:

- Primary color: Dark gray (#333333) for a professional, modern base.
- Background color: Light gray (#E0E0E0) to ensure contrast with the content.
- Accent color: Teal (#008080) for highlights and interactive elements.
- Font: 'Inter', a sans-serif font for a modern and readable look, used for both headings and body text.
- Grid-based layout for consistent and responsive design across devices.
- Subtle transitions and hover effects for enhanced user experience.
- Simple, geometric icons to complement the sleek design.
